# 📈 AI-Based IPO Performance Prediction using Deep Learning

> Research Internship Project | Indian Institute of Management Mumbai

## Overview

Initial Public Offerings (IPOs) are influenced by a complex interaction of company fundamentals, market sentiment, macroeconomic conditions, and investor behaviour. Accurately predicting IPO listing performance is a challenging financial forecasting problem due to the large number of interconnected variables.

This research project develops an end-to-end Artificial Intelligence pipeline for predicting IPO listing performance by integrating structured financial data, macroeconomic indicators, valuation metrics, and company-specific characteristics using Machine Learning and Deep Learning techniques.

The project was carried out during my Research Internship at the **Indian Institute of Management Mumbai**, focusing on quantitative financial analytics and predictive modelling.

---

# Objectives

- Develop a comprehensive IPO prediction framework using Artificial Intelligence.
- Build structured datasets from multiple financial data sources.
- Identify the most influential variables affecting IPO performance.
- Develop predictive models using Deep Learning.
- Evaluate model performance using statistical validation techniques.
- Support quantitative investment decision-making through predictive analytics.

---

# Problem Statement

IPO performance depends on numerous financial and economic variables including:

- Company Financial Statements
- Revenue Growth
- Profitability
- Market Capitalization
- IPO Subscription
- Valuation Metrics
- GDP Growth
- Inflation
- Interest Rates
- Market Volatility
- Industry Trends

Traditional valuation methods cannot effectively capture these nonlinear relationships.

This project applies Machine Learning and Deep Learning to model these complex interactions.
